Yes! Firefox OS will be released in 2013! It is a complete HTML5/Web based OS for mobile phones.
It treats all apps like dial screen, app menu, games, appstore as web applications. It's based on top of Linux (just like Android) and hopefully will bring a change in the monopoly by Android for being the only great, Open Source and free OS for mobiles/tabs/even netbooks. More than 50% of Android apps are web based, so it is very easy to port them on Firefox OS. Aren't you excited? Our loved company which gave us a superb free browser, an awesome e-mail client, is now launching its OS! Want to run it on your desktop/notebook? You actually can! These steps are targeted for Ubuntu users, Mac and Windows users can also achieve it similarly 1. Download the latest nightly release of b2g from here ftp://ftp.mozilla.org/pub/mozilla.org/b2g/nightly/latest-mozilla-central/ (try http if ftp doesn't work) 2. Extract the bz2 file. (you will get a b2g executable but it would show a black screen) 3. Clone the gaia repository from git in your b2g extracted folder, using this command in your b2g folder -
git clone git://github.com/mozilla-b2g/gaia
4. Create a gaia profile using this command in your b2g folder
make -C gaia profile
5. Finally running Firefox OS is done by this command in your b2g directory. ./b2g -profile gaia/profile
