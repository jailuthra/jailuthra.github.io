  • An Operating System is like Windows, or Android, iOS, Mac OSX etc.
  • Most special of them are GNU/Linux distros , Ubuntu is one of them.
  • Many of you will think that why should you learn using a whole new operating system. But you have to do that even in your current OS (assuming it is Windows). With Windows 8 coming soon, you will have to learn a whole new thing, so it is better to learn something good rather than wasting your time and money on crap.

  • Ubuntu 12.04 is packed with amazing features, I have written a review on it.
  • It also has a great app Gwibber, which keeps you updated (via Notifications) with your Social Networks like Facebook, Twitter etc.
  • Ubuntu One is a very great app for sync services (like Dropbox), it is integrated in the ubuntu panel.
GNU/LINUX
  • Linux is a kernel i.e. it is the is the bridge between applications and the actual data processing done at the hardware level. Linux is not an operating system without applications.
  • A Linux distro like Ubuntu, Fedora, Mint; truly makes an OS with the help of the Linux Kernel and applications.
  • With so many distros to choose from you can always find a distro which suits you. "Your OS should suit you and not vice-versa. " - Someone Like Me
  • In Ubuntu (and other Linux distros) you cannot run the usual .exe files which you may be using earlier (if you used windows), for installing software in Ubuntu you need to have its .deb package or its source-code.
  • .deb files can be installed very easily using a package manager like Ubuntu Software Center or Synaptic.
  • If you are desperate to run .exe files in Linux you can use Wine for GUI applications and DosBox for DOS apps. ( I recommend you to leave your previous .exe world completely)
  • You DO NOT need an anti-virus for Ubuntu as Linux itself is Virus-free (atleast for now, maybe some years ahead you have to).
OPEN SOURCE (FREE AS IN FREEDOM)
  • Free software is a matter of the users' freedom to run, copy, distribute, study, change and improve the software.
  • It is open source, which means that you can modify it or improve it's source code, which is freely available to every user.
  • Most of the software which come preinstalled in Ubuntu are also open source, like Firefox, Libre Office etc.
  • There is also freedom to choose between so many Desktop Environments and Default Applications.
  • Those who prefer using Google Chrome can download Chromium (Chrome's open source brother) from Ubuntu Software Center. Or you can download Google Chrome itself from the web.
  • If you are used to the desktop interface that Windows offers, you can use KDE which feels much more similar to Windows.
FREE AS IN BEER
  • Here free refers to cost-free. You DO NOT have to pay a single dime for using, upgrading or downloading free software on Ubuntu.
  • Ubuntu is and always be, free of cost.
  • It saves you so much money (or helps you from committing the crime of piracy) because you don't need to spend (waste money) on buying an OS for your computer.
